## Setup

- `python -m venv .venv`
- `pip install -r requirements.txt`

## Dependencies

- `brew install pandoc`
- `brew install librsvg`
- `brew install --cask basictex`
- `sudo tlmgr update --self`
- `sudo tlmgr install collection-fontsrecommended`
